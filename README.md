# ⚡ VSCode-Extension-Downloader
一个小巧的 **纯前端工具**，帮助你从 **VSCode Marketplace** 快速提取 `.vsix` 安装包的直链，无需登录即可下载并离线安装。

---

## 🚀 为什么用它？
在局域网或无网络环境的电脑上 **手动离线安装 VSCode 插件** 时，手动拼接 `.vsix` 下载链既费时又容易出错。

**解决方案**：  
只需要把插件市场里的一段信息复制进这个前端页面，它就会立即给出 **微软官方直链**，支持一键 **下载 / 复制**。

---

## 🏗️ 技术栈
- [Vite 7](https://vitejs.dev/) + [Vue 3](https://vuejs.org/) (Composition API)  
- [TailwindCSS 3](https://tailwindcss.com/) — 无需一行自定义 CSS  
- 纯前端实现，零后依赖；可直接托管在静态网站（GitHub Pages / Netlify / Vercel）。
